1. このコードをコピーして"YOUR_URL_HERE"の部分に目的URLを代入する。

```JavaScript
var hereURL = window.location.href;
var correctURL = "YOUR_URL_HERE";
if (hereURL != correctURL) {
  alert("Here is a fake site.");
  location.href = correctURL;
}
```
